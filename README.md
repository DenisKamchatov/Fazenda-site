# Fazenda-Django-site

Django website for Fazenda Cottage Village

## Versions
1) Python - 3.9.5
2) Pip - 21.1.3
3) Django - 4.0.4

## Build Setup

```bash
# install dependencies
$ pip install -r requirements.txt

# install migrations
$ py manage.py migrate

# serve with hot reload
$ py manage.py runserver
